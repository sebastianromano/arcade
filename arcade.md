Arcade Games

- [x] Pong: A two-player table tennis-like game where players control paddles to hit a ball back and forth.
- [x] Tetris: A puzzle game where players manipulate falling blocks to create complete lines.
- [] Breakout: A single-player game where the player controls a paddle to bounce a ball and break bricks at the top of the screen.
- [] Pac-Man: A maze game where the player controls Pac-Man to eat dots while avoiding ghosts.
- [x] Space Invaders: A shooting game where the player controls a spaceship to defend against descending alien invaders.
- [] Asteroids: A space-themed game where the player controls a spaceship to shoot and destroy asteroids and flying saucers.
- [x] Flappy Bird: A side-scrolling game where the player controls a bird, attempting to fly between columns of green pipes.
- [] Minesweeper: A single-player puzzle game where the player must clear a board without detonating any hidden mines.
- [] Tic-Tac-Toe: A simple two-player game where players take turns marking X's and O's on a 3x3 grid.
- [] Memory Game: A card-matching game where players flip over cards to find matching pairs.
- [] Whack-a-Mole: A game where players use the mouse to click on moles that pop up from holes.
- [] 2048: A sliding tile puzzle game where players combine numbered tiles to reach the 2048 tile.
